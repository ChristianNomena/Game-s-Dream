Le HTML

HTML (Hypertext Markup Language)
C'est un langage de balisage concçu pour présenter les pages web. Il sert à écrire de l'Hypertexte, d'où son nom.
  1. Les bases en HTML

    1. Elément, balises, attributs

      2. Un élement est constitué d'une balise ouvrante, d'une balise fermante et d'un contenu.

    2. Structure

      3. Après avoir ouvert une balise, il faut la fermer.

    3. Affichage

      4. Il faut bien définir le type et l'encodage du fichier : type html et encodage utf-8

    4. Identation et commentaire

      5. Pour la compréhension du code, il faut penser à l'indentation et aux commentaires.

    5. Les listes

      1. Listes ordonnées

        2. Les éléments d'une liste ordonnée sont numérotés.

      2. Listes non-ordonnées

        3. Ce sont des listes qui n'ont pas de notion d'ordre, elles sont aussi connues sous le nom de liste à puce.

    6. Les liens

      • Il faut utiliser l'élément href (hypertext reference), il va prendre comme valeur la cible du lien.
      • La cible du lien est tout simplement l'adresse de la page de déstination du lien.

      1. Liens externes

        2. Un lien externe est un lien qui part d’une page d’un site et ramène les utilisateurs vers une autre page d’un autre site.

      2. Liens internes

        3. Un lien interne permet de se déplacer dans un même document ou alors de se déplacer dans un point précis d'un autre document ( à condition qu'il est une ancre interne)

      3. Autres types de liens

        4. Les ancres (pour se déplacer dans une page), pour envoyer un mail et pour afficher ou télécharger un fichier.

Le CSS

  1. Emplacements
    • fichier .css (la plus recommandée)
    • Dans le head de l'HTML
    • Par l'attribut style (la moins recommandée)

  2. Sélecteurs
    • Sélecteur universel: "*"

      • Sélectionne toutes les balises sans exceptions

    • Sélecteurs simple
      • Sélecteur d'élément

        • exemples: p, h1,...

      • ".Class"

        • Peut sélectionner plusieurs balises

      • "#Id"

        • Unique à une balise

    • Sélecteurs combinatoires
      • Sélecteur déscendant: "A B"

        • Balise contenue dans une autre

      • Sélecteur de balise qui suit une autre: "A+B"
      • Exemple
        • ceci est un balise

        Ce mot en gras est déscendant du paragraphe et aura un fond rouge
        Toute cette paragraphe suit la balise h6 et sera en bleu

    • Sélecteur d'attribut

      • Balise qui contient un attribut spécifique

    • Exemple

      • Les paragraphes qui possèdent un titre auront un fond jaune


  3. Pseudo-classes ":"

    4. Surtous utilisées sur les liens

    • :link

      • Un lien qui n'a pas encore été visité

    • :visited

      • Indique que le lien a déjà été visité

    • :hover

      • Lorsque la souris est au-dessus du lien

    • :active

      • Lorsqu'on clique sur le lien

    • :first-child

      • Sélectionne le premier enfant de la balise

    • :last-child

      • Sélectionne le dernier enfant de la balise

    • :nth-child()

      • Cible certains enfants d'un élément HTML qui répondent aux conditions mises entre parenthèses

    • Exemple:
      • Cliquez ici

  4. Pseudo-éléments "::"

    5. Permet de modifier juste une partie

    • ::first-letter

      • La première lettre d'un texte dans l'élément

    • ::first-line

      • La première ligne d'un texte dans l'élément

    • ::selection

      • La partie sélectionnée par l'utilisateur

    • ::before

      • Insérer un contenu avant l'élément

    • ::after

      • Insérer un contenu après l'élément

    • Exemple

      • Ceci est un paragraphe avec plusieurs ligne de texte
      Dont ceci est la deuxième ligne et si vous le sélectionnez, elle devient vert


  5. Les préfixes vendeurs

    6. mots clefs optimisant artificiellement la compatibilité de certaines propriétés CSS avec certaines versions de navigateurs

    • Safari et Chrome: "-webkit-"
    • Firefox: "-moz-"
    • Internet Explorer: "-ms-"
    • Opéra: "-o-"

AJAX

AJAX est l'acronyme d'Asynchronous JavaScript And XML, autrement dit JavaScript Et XML Asynchrones.

L'idée même d'AJAX est de faire communiquer une page Web avec un serveur Web sans occasionner le rechargement de la page. C'est la raison pour laquelle JavaScript est utilisé, car c'est lui qui va se charger d'établir la connexion entre la page Web et le serveur.

Les outils necessaire pour executer AJAX sur un page web sont:

  • Fichier html
  • Fichier xml pour les donné au niveau du server par exemple
  • Fichier js pour Instancier l’objet XMLHttpRequest

JQUERY

JQuery est une bibliothèque JavaScript. En d'autres termes, un fichier d'extension .js.Utilisé pour la facilité du code javascript

Puisque c'est une bibliothèque sont utilisation necessite la telechargement du fichier jQuery.js ou jQury.mini.js

jQuery repose sur une seule et unique fonction : jQuery(), ou son alias, $(). Cette fonction accepte un ou plusieurs paramètres et retourne

  • Une fonction, qui sera exécutée dès que le DOM est disponible. 
    • Exemple:
                                    $(function() {
                                        $(".rouge").fadeOut("slow",function(){
                                            $(this).fadeIn("slow");
                                            }); 
                                        });
  • Un sélecteur CSS

    • Exemple : $('.rouge').css('background','red');

  • Un élément HTML

Le sélecteur jQuery $('sel') utilise la syntaxe CSS. En remplaçant sel par un sélecteur CSS quelconque

Exemples : 
                                        $('a') sélectionne tous les liens hypertextes ;
                                        $('.rouge') sélectionne les éléments de classe rouge ;
                                        $('#e2') sélectionne l'élément d'identifiant e2 ;
